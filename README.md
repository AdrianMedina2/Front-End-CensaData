# CensaData

Aplicación web desarrollada con HTML, CSS y JavaScript puro con Bootstrap para la estructura.
El objetivo es describir y dar a conocer CensaData, una aplicación dirigida a la gestión y análisis de información censal histórica del INIDE.

## Tecnologías usadas
- HTML5
- CSS3
- JavaScript
- Bootstrap (UI y estilos)
- Visual Studio Code
- GitHub

---

## Instalación

### Requerimientos
- Navegador moderno (Chrome, Edge, Firefox, etc.)
- Editor de código (opcional, recomendado VS Code)

### Setup Frontend
1. Clonar el repositorio:
   ```bash
   git clone https://github.com/AdrianMedina2/Front-End-CensaData.git
   cd Front-End-CensaData
1. Abrir el proyecto en tu editor o directamente en el navegador:
   - Si usas Visual Studio Code, preferiblemente instalar Live Server para levantar el servidor local.
   - O en su defecto, abrir el index.html en tu navegador.
